# Customer Segmentation Project

## Overview
This project uses K-Means Clustering to segment customers based on annual income and spending score.

## Technologies Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Scikit-learn

## Features
- Customer data analysis
- K-Means clustering
- Customer segmentation visualization

## Output
The project groups customers into different clusters based on spending behavior and income patterns.

## Algorithm
K-Means Clustering

## Dataset
Mall Customers Dataset
